# 🎮 Memória Pokémon

**Memória Pokémon** é um jogo da memória minimalista inspirado em Pokémon. Teste sua mente, suba de nível e bata recordes!

---

## 🚀 Funcionalidades Principais
* **Dificuldade Progressiva:** A cada nível, mais pares são adicionados (máximo de 18 cartas).
* **Persistência de Dados:** Recorde e configurações salvos automaticamente no seu dispositivo via `localStorage`.
* **Feedback Imersivo:** Vibração tátil (Capacitor Haptics) ao errar e sons originais dos Pokémon (Cries). ``**em progresso**``
* **Visual Claro:** Interface otimizada para manter a nitidez e animações 3D de flip.
* **Modo noturno**

## 🛠️ Tecnologias
* **Core:** React.js (Hooks customizados)
* **Mobile:** Capacitor (Haptics API)
* **Estilo:** CSS3 (3D Transforms & Flexbox)
